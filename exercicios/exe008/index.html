Principais fomatações

Negrito / Destaque

Texto em negrito sem semântica, usando <B> </B>

Texto em negrito com semântica, usando <strong> </strong>

Itálico / Ênfase

Texto em itálico sem semântica, usando <i> </i>

Texto em itálico com semântica, usando <em> </em>

Texto marcado

Texto marcado como um marca texto, com semântica, usando <mark> </mark>

Texto grande e pequeno

Texto grande utilizando <big> </big>, não semântica (tag obsoleta), e texto pequeno utilizando <small> </small>, com semântica

Texto deletado

Utilzado para indicar a exclussão de texto do texto original usando <del> </del>, com semântica

Texto inserido / Sublinhado

Texto inserido ao texto original, usando <ins> </ins>, com semântica

Mesmo resuldado visual utilizando a forma, usando <u> </u>, sem semântica

Texto sobrescrito / Texto subscrito

Texto meio caractere a cima da linha original (Ex: X20 + 15), usando <sup> </sup>, com semântica

Texto meio caractere a baixo da linha original (Ex: H2O), usando <sub> </sub>, com semântica
